# Bank Account System

**Автор:** Малкина Софья

## Описание
Консольное банковское приложение ддля управления счетами с поддержкой:
- Разных типов счетов( Расчётный, Сберегательный, Кредитный )
- Истории транзакций в виде очереди
- Перевода средств между счетами
- Сохранения и загрузки данных в JSON

## Установка и запуск
```bash 
# Клонирование репозитория
git clone https://github.com/SonyaMalkina/Bank-Account-System.git
cd bank_account_system

# Запуск
python main.py
```
